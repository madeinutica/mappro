Mapro Solar Projects Map - Embed Demo

Embedded Map Example

This demonstrates how to embed the Mapro solar projects map on any website using the embed script.

How to Use

1. Include the Embed Script

Add this script tag to your HTML page:

<script src="https://your-domain.com/mapro-embed.js"></script>

2. Add a Container Element

Add a div element where you want the map to appear:

<div id="my-map"></div>

3. Initialize the Map

Call the embed function when the page loads:

<script> // Basic usage createMaproMap('my-map'); // With custom options createMaproMap('my-map', { height: '600px', center: [-75.5, 42.7], zoom: 8, showHeader: false }); </script>

Configuration Options

createMaproMap(containerId, options); // Options: { height: '500px', // Map container height showHeader: false, // Show/hide "Mapro Solar Projects" header (default: false) center: [-75.5, 42.7], // Initial map center [lng, lat] zoom: 7 // Initial zoom level }
Note: Make sure to host the mapro-embed.js file on your web server and update the script src URL accordingly. The script requires internet access to load Mapbox GL JS and connect to the Supabase database.
